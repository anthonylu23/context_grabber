Context Grabber

This installer will set up:

  • ContextGrabber.app in /Applications
  • cgrab CLI in /usr/local/bin

After installation:

  • Open ContextGrabber.app from Applications to start the menu bar agent.
  • Use cgrab --help in your terminal for CLI usage.
  • Run cgrab doctor to verify your setup.

The app will request Accessibility and Screen Recording permissions on first use for desktop capture.
